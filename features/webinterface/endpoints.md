# Endpoints

An **endpoint** is a residential proxy slot. Each board you want to earn with needs
an endpoint assigned to it.

## Managing endpoints

- **Buy** endpoints from the dashboard.
- **Assign** an endpoint to a board (in the appliance, assign endpoints to the
  discovered boards).
- **Unbind** an endpoint before re-assigning it — endpoints assigned to the legacy
  agent must be unbound first.
- **Rotate** a board's exit IP when needed.

![The Proxies page listing each board's residential endpoint with its country, exit IP (redacted here), risk score and monthly cost.](/images/features/proxies-endpoints.png)

## Pricing

Endpoint pricing may change as UNetwork traffic grows. Any change applies to
renewals and new endpoints and is announced in advance in the dashboard.

Endpoints must carry **UNetwork traffic only** — see [Traffic Usage](/rules/traffic_usage).
